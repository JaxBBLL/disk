# disk

一个的简单云盘，可以用来局域网传输文件

## 配置文件

config.json

```js
{
  "dest": "D:/html/release", // 根目录所在，绝对路径或相对路径
  "hasDel": true, // 是否能删除
  "port": 3000 // 端口
}
```
